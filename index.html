組員:廖翊程、魏麟驊、蔡松穎、李佳燁、吳沁怡、張傑祐、鄭雅云、石欣禾、陳子龍

專題目的與動機

本專題名為《靜界碎片之謎》，致力於打造一款結合校園場景與互動技術的沉浸式解謎遊戲。 玩家將穿梭於神秘的靜界世界中，透過解謎、對話與任務挑戰，深入了解校園文化與技術應用，寓教於樂。

遊戲結合了多種技術，包括 Unity 遊戲引擎、聊天機器人（NLP）、影像辨識（CNN模型），以 Python 和 C# 為開發語言，並運用多款工具輔助開發。 我們希望讓玩家在輕鬆互動的氛圍中，認識人工智慧的應用方式，同時提升學習興趣與實作能力。

團隊在專案開發過程中，從場景設計到程式邏輯的構建，持續優化互動性與遊戲體驗，最終完成一款既有趣又具啟發性的作品。
