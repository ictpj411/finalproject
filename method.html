研究方法與步驟

研究架構

本專題整合「聊天機器人」與「影像辨識系統」於 Unity 遊戲中，透過 Python 開發對應模型，並以 C# 負責遊戲邏輯控制與畫面互動，達成跨平台技術整合。

聊天機器人流程

聊天機器人部分，我們採用 Naive Bayes 模型，對語料數據進行預處理（斷詞、去除停用詞、向量化），並以 90% 訓練集、10% 測試集進行建模。 為提升對話真實感，當模型無法理解玩家輸入時，設計預設回應以避免冷場。

影像辨識流程

影像辨識系統使用 ResNet-18 深度學習模型，資料集涵蓋多場景、多時段與不同天氣條件，提升模型泛化能力。影像預處理包含尺寸壓縮 (80x80)、格式統一，並設置 50 個訓練週期。

模型訓練完成後，我們將結果整合到遊戲流程中，玩家透過上傳現場拍攝的圖片，完成任務挑戰並推進劇情。

整體研究步驟

  • 🔍 確定專題方向與技術需求
  • 🧩 資料收集與預處理（圖片 + 對話數據）
  • 🛠️ 模型訓練與測試（聊天機器人 + 影像辨識）
  • 🖥️ 遊戲開發與技術整合
  • 🧪 測試與優化（改善辨識率與對話流暢性）
  • 🚀 部署與發表展示
